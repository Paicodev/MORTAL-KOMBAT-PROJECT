# MORTAL KOMBAT WORLD CUP MANAGER

Este simulador mezcla dos mundos diferentes que me gustan muchos, se trata de un gestor de torneos del videojuego Mortal Kombat (Especificamente una version M.U.G.E.N de este),
y el formato que utiliza la Copa del Mundo de Futbol (con eliminatorias y el torneo de 32 equipos).
En este caso, se utilizaran a los luchadores del videojuego en lugar de los paises, por lo tanto, se dividirán también por confederaciones.

## ⚙️ Características
* **Motor de Simulación Híbrido:** Partidos interactivos (Humanos) o simulados por CPU basados en estadísticas.
* **Sistema de Ligas y Playoffs:** Algoritmo Round Robin para eliminatorias (Fase de Grupos) y generación dinámica de brackets para la fase final.
* **Dashboard Web Dinámico:** Generación en tiempo real de archivos HTML con CSS para visualizar tablas, fixtures y el cuadro del torneo.
* **Persistencia de Datos:** Sistema de guardado con múltiples slots, historial de campeones y Ranking Mundial (estilo Ranking FIFA) persistente.

## 🛠️ Tecnologías
* Java 17+
* Programación Orientada a Objetos (POO)
* Estructuras de Datos y Colecciones (List, Map)
* File I/O & Serialización
